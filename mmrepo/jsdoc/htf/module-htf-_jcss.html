(static) _jcss(stylesheet) → {String}

爪哇腳本樣式表： 將爪哇腳本編寫的程式碼轉換成樣式表格式。
Parameters:
Name Type Description
stylesheet Object 爪哇腳本的樣式表。 是 `String` 與 `Object` 組成的物件陣列。
Source:
  • htf.js, line 692
Returns:
若樣式表格式不符則回傳 `''` 空文字。
Type
String
Example
_jcss( {
    'body': {
        margin: '0',
    },

    '.TxDiv': {
        width: '100%',
        height: '100%',

        '&_aLink': {
            display: 'block',

            'body.esPc &': {
                display: 'none',
            },
        },
    },
} );

// CSS
// body {
//     margin: '0';
// }
// .TxDiv {
//     width: '100%';
//     height: '100%';
// }
// .TxDiv_aLink {
//     display: 'block';
// }
// body.esPc .TxDiv_aLink {
//     display: 'none';
// }
