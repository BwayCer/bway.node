初始化標記標籤。

markHel 規範：
* 若 `isPureText === true`：

```
  main: 放於身體段落的文字,
  script: 用於設定元素標籤之程式碼 }
```
* 若 `isPureText === false`： 
```
{ main: 主要輸出元素標籤（最後執行的）,
  ...key: 追蹤之元素標籤 }
```
